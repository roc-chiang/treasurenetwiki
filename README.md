# TreasureNet - wiki

##### A modern, lightweight and powerful wiki app built on NodeJS


## Getting Started

- Clone this project.
- ```docker-compose up -d ``` 

## Docker Hub

https://hub.docker.com/repository/docker/treasurenet/wiki

## Build via dockerfile

```docker build -t treasurenet/wiki:1 -f dev/build/Dockerfile .```
